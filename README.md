# Employee-Management

## Pre-requisites
- Python
- Python IDE
- MySql Workbench

## Getting Started
- git clone <repo_name>
- python -m venv Employee
- Employee\Scripts\activate
- pip install -r requirement.txt
- streamlit run main.py
- python backend.py
